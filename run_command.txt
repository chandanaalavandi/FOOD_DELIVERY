Open PowerShell and run these in order:

Start MySQL (if not running):
 net start MySQL80
Start Tomcat:
 $env:CATALINA_HOME = 'C:\tomcat9'; & 'C:\tomcat9\bin\startup.bat'
Wait 10 seconds, then open browser:
http://localhost:9000/food-delivery/

To Stop Everything
# Stop Tomcat
$env:CATALINA_HOME = 'C:\tomcat9'; & 'C:\tomcat9\bin\shutdown.bat'

# Stop MySQL
net stop MySQL80
